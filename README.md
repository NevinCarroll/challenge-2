# Challenge 2: Montana County Lookup Extended
This program allows the user to look up the 
county information that is pertained to a 
license plate prefix. It runs in the console
and reads and stores county data in a csv file

## Requirements:

Python 3.12

## Libraries:

csv

## How to run:
Just run the main.py file while you are in
the directory:
```
python main.py
```
